# Feed Application


## Feed creation
- a user can create a feed
- a feed should be of a certain length

## Feed update
- a user can update a feed
- the last updated feed should be saved
- the timestamp of last updated feed should be retained

## Feed deletion

- a user can delete a feed
- fee deletion should be soft delete

## Feed List

- a user should be able to see feeds
- a user should be able to see feeds only of
the people they have followed.

## Follow a User

- a user should be able to follow another user

## Maintaining Followers
- a user should be able to see who their followers are


## Registration
- a user should be able to register/login in the application
